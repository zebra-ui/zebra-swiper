TIP

v3版本所有平台都可通过v-on绑定事件并获取swiper实例。

swiper ​

swiper组件挂载完成时触发。

html
<z-swiper @swiper="onSwiper">
  <z-swiper-item v-for="item in list" :key="item.id">
    <DemoItem :text="item.text"></DemoItem>
  </z-swiper-item>
</z-swiper>
vue
<script setup>
const onSwiper = (swiper) => {
  console.log('swiper实例：', swiper)
}
</script>
